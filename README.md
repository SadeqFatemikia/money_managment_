# 💰 Money Management Script

A simple and practical Python script for managing and allocating your monthly budget using different saving strategies.

## 🎯 Purpose

This tool helps you manage your monthly income by allocating it across different categories (Main Account, Shopping & Services, Investing & Savings) based on your preferred strategy.

## ✨ Features

- Two budget allocation strategies:
  - **50/30/20**: 50% Main Account, 30% Shopping & Services, 20% Investing & Savings
  - **80/20/20**: 80% Main Account, 20% Shopping, 20% Investing & Savings
- Takes user name and total income as input
- Automatically calculates and displays exact amounts
- Saves output to a text file
- Error handling for invalid inputs
- Simple command-line interface

## 🛠️ Requirements

- Python 3.8 or higher

## 🚀 How to Run

1. Download the `money_manager.py` file.
2. Open your terminal or command prompt.
3. Run the following command:

```bash
python money_manager.py
